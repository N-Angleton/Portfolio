The Gram

The Gram is a full stack clone of the image sharing site Instagram. Using a Rails backend and a PostgreSql database, it replicates the CRUD functionality behind following users, sharing images, writing comments, and liking posts. Additionally, using a React/Redux frontend, it emulates the minimalist layout and color palette of the original site.

WeGo

WeGo is a social media web app for meeting new people and trying new things. It was built in collaboration with three other engineers using the MERN stack. Its backend uses Express, run via a Node.js environment; storage is handled by MongoDB and AWS S3; and its frontend uses React and Redux.

Etchify

Inspired by traditional etchings, the site transforms user uploaded images through a combination of line recognition and perceived luminance matching. It employs JavaScript, the Canvas API, and whole lot of pixel math.
